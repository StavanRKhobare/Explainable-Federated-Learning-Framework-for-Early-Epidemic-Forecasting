Introduction
Explainable Federated Graph Neural Network for Epidemic Forecasting is an AI-based system designed to predict dengue and malaria outbreak risk using district-level disease trends, weather conditions, and spatial relationships between regions.
The project combines temporal learning and graph neural networks to understand how outbreaks evolve over time and spread across neighboring districts. A split federated architecture is used where local clients process temporal data and generate embeddings, while a central graph model analyzes spatial disease spread and predicts outbreak risk.
In Phase 2, the project progressed from system design to implementation by developing the data pipeline, graph-based prediction model, inference engine, and interactive dashboard for district-level outbreak visualization and analysis.

Slide No. 5
Project Overview
The proposed system is a spatio-temporal epidemic forecasting framework developed to predict disease outbreak risk at the district level. The project focuses primarily on dengue forecasting using historical case trends, weather conditions, population statistics, and spatial relationships between neighboring districts.
The architecture is divided into two major components:
Client-side temporal processing, where local district data such as previous case counts, rainfall, and temperature are analyzed to learn outbreak trends over time.
Server-side spatial graph learning, where districts are represented as nodes in a graph and neighboring districts are connected through edges to model geographical disease spread.
The system combines temporal embeddings from local clients with a Graph Neural Network (GNN) at the central server to generate outbreak predictions and hotspot alerts. The project also includes an inference pipeline and an interactive dashboard for visualization, explainability, and district-level outbreak analysis.

Model Training Behaviour
Overall Learning Performance
The training loss decreases steadily across epochs, indicating stable learning and successful optimization of the model.
Validation loss remains relatively low and stable, suggesting that the model is generalizing well without severe overfitting.
Regression Performance (MAE & RMSE)
MAE and RMSE gradually decrease during training, showing improvement in predicting outbreak trends and disease intensity.
Lower regression error indicates that the model is learning temporal disease progression effectively.
Outbreak Classification Performance (AUC-ROC)
The model achieves validation AUC values close to 0.96, demonstrating strong capability in distinguishing outbreak and non-outbreak districts.
High AUC values indicate effective spatial and temporal pattern learning.

Precision-Recall Performance (AUPRC)
AUPRC improves progressively during training, showing better identification of outbreak-positive districts despite class imbalance.

F1-Score Analysis
F1-score shows gradual improvement throughout training.

Conclusion
The project successfully demonstrates a spatio-temporal epidemic forecasting framework capable of predicting district-level outbreak risk using Graph Neural Networks and temporal learning techniques.
By integrating disease trends, weather conditions, and spatial district relationships, the system is able to learn meaningful outbreak patterns and generate hotspot predictions with strong classification performance.
The implementation of inference pipelines, explainability modules, and interactive outbreak visualization transforms the project from a conceptual architecture into a functional AI-driven epidemic intelligence prototype.
The project also establishes a strong foundation for future development in decentralized healthcare AI, real-time outbreak monitoring, and scalable public health forecasting systems.
